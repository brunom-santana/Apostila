Introdução ao HTML e CSS

HTML e CSS são as duas linguagens de programação básicas para a criação de páginas web. HTML é responsável pela estrutura e organização do conteúdo, enquanto CSS é usado para estilizar e formatar o layout da página. HTML usa estruturas de marcação para identificar e definir diferentes elementos de uma página, como títulos, parágrafos, imagens e links. CSS é usado para controlar a aparência dos elementos HTML. Usando CSS, você pode definir cor, fonte, tamanho, espaçamento e posicionamento dos elementos para criar layouts atraentes e responsivos. CSS também oferece recursos avançados, como animações, transições e efeitos visuais. Em resumo, o HTML define a estrutura e o conteúdo da página, enquanto o CSS controla o estilo e a aparência. Juntos, eles são fundamentais para a criação de páginas da web modernas e funcionais.

Um exemplo de Tag HTML, seria a 'p' de parágrafo, ou a "header" de cabeçalho. De maneira geral, toda página web HTML, por mais simples que seja, necessita de uma estrutura composta por diversas tags que compõe a estrutura dela. Por exemplo:

  • (html) : É a tag que identifica um documento como HTML;
  • (head) : É a tag que apresenta as informações gerais da página;
  • (title) : É a tag que informa o título da página na barra do navegador web;
  • (body) : É a tag que delimita o conteúdo todo da página web.

Em resumo, o HTML define a estrutura e o conteúdo da página, enquanto o CSS controla o estilo e a aparência. Juntos, eles são fundamentais para a criação de páginas da web modernas e funcionais.

Tags de Formatação

Existem tags que podem ser utilizadas para apresentação de texto em uma página web. Por exemplo:

  • < p > : Tag para a utilização de um parágrafo;
  • < span > : Tag para representar um texto sem significado.

    • A tag de span deve ser utilizada quando nenhum outro elemento de texto semântico representar um significado adequado.

Tags de Texto

Alguns exemplos de tag de Texto:

  • < small > : Texto pequeno;
  • < sub > : Texto subscrito;
  • < em > : Texto enfatizado;
  • < sup > : Texto sobrescrito;
  • < strong > : Texto importante;
  • < b > : Texto em negrito.

Atributos

Atributo id

Um atributo id serve para identificar um único elemento de uma página web. Este atributo é global, então, qualquer tag do HTML pode possuir um id. Seu valor deve começar com uma letra ou underline. Um elemento não pode possuir mais de um atributo de id. Um atributo de ID pode auxiliar o CSS e até mesmo o JavaScript.

Atributo class

Este atributo serve para dar categoria a um grupo de elementos que possuem o mesmo comportamento. Assim como o atributo de id, o atributo class também é global, ou seja, qualquer tag do HTML pode possuir uma class. Seu valor também começa com uma letra ou underline, e também auxilia tanto o CSS quanto JavaScript

Div

A divisória permite o agrupamento de um conjunto elementos em uma caixa block-level. Cada conteúdo presente em uma divisória será apresentado em uma nova linha, e este comportamento pode ser alterado utilizando o CSS. A div melhora a organização do código da sua página web, além de ser permitido colocar qualquer elemento dentro de uma divisória.

Nav

A Nav é utilizada para representar um agrupamento de links de navegação. Por exemplo: Paginação de uma busca.

Header

Header é utilizada para representar o cabeçalho de um documento ou de uma seção.

Section

Esta tag representa uma seção dentro de um documento, esta tag é utilizada para descrever separadamento as seções e tópicos de um certo documento.

Main

A tag é utilizada para especificar todo o conteúdo principal, ou de maior relevância, de um documento. Uma página HTML deve conter apenas um conteúdo principal, utilizando a tag Main. Por exemplo: O conteúdo de um site.

CSS

CSS, na verdade, é uma sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é simples de se aprender, e é comumente utilizado com as linguagens HTML e XHTML. Criado em 1996 pelo World Wide Web Consortium com o intuito de auxiliar o HTML, pois a linguagem HTML não foi projetado para ter tags que ajudariam a formatar a página. O estilo CSS sempre é definido em um arquivo CSS separado da página HTML, assim, tornando mais fácil o desenvolvimento da página web. Essa separação permite manter o código mais organizado e menos confuso para o programador, permitindo que o conteúdo seja mostrado de diversas maneiras diferentes, adaptando-se a diferentes tipos de dispositivos e tamanhos. De maneira geral, o CSS é fundamental na criação de páginas web, dando uma experiência mais agradável para o desenvolvedor.
